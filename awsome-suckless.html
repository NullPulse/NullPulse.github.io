Awesome-Suckless

Browser

  • min: A fast, efficient, and privacy-focused web browser with a minimal user interface.
  • qutebrowser: A keyboard-driven web browser with a minimal GUI and a vim-like interface.
  • surf: A simple web browser based on WebKitGTK with a minimalistic design.
  • w3m: A text-based web browser with support for tables, frames, color, and more.

Development & Programming

  • Suckless Compiler Suite (SCC): A simple, efficient, and easy-to-understand compiler suite adhering to the Suckless philosophy.
  • stagit: A static Git page generator, simple and lightweight.
  • farbfeld: A simple image format to represent RGB, RGBA, and alpha images.
  • gcc: The GNU Compiler Collection for various programming languages.
  • vim: A highly configurable text editor for efficient text editing.

Display Managers

  • Ly: A TUI (Text-based User Interface) display manager with a focus on simplicity and ease of use.
  • Slim: A simple login manager with a minimalist design.

File Managers

  • Double Commander: A cross-platform open-source file manager with two panels (GUI).
  • lf: A terminal file manager written in Go (CLI).
  • nnn: A fast and resource-friendly file manager with an enhanced terminal user interface (CLI).
  • PCManFM: Lightweight file manager for LXDE (GUI).
  • ranger: A console file manager with VI key bindings (CLI).
  • Thunar: Xfce file manager offering a clean and easy-to-use interface (GUI).

Terminals

  • rxvt-unicode (urxvt): Terminal emulator with support for Unicode and client-server architecture.
  • ST (Simple Terminal): Lightweight and simple terminal designed to be easy to understand and extend.
  • xterm: Classic terminal emulator for the X Window System.

Graphics & Imaging

  • farbfeld: A simple image format to represent RGB, RGBA, and alpha images.
  • sxiv: A simple X image viewer with a focus on keyboard shortcuts and efficiency.

Multimedia

  • mpv: A free, open-source, and cross-platform media player with a focus on simplicity and performance.
  • ncspot: A command-line Spotify client written in Rust and inspired by ncurses.
  • moc: Music On Console (MoC) is a console audio player for Unix-like systems.
  • Siren: Lightweight notification daemon for multimedia keys.
